`pip install django djangorestframework`


## NPM
`npx create-react-app appname`
`cd appname`
`npm start`
### Installation
`npm install react-router-dom`
`npm install @material-ui/core`
`npm install @material-ui/icons`
`npm install @mui/icons-material`
